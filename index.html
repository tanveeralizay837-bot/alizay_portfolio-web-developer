Final Year Project

Virtual Marketplace for Local Artisans

A full-stack e-commerce platform connecting Pakistani artisans with buyers — built with real security, real transactions, and real role-based access control.

Full Stack Web Application

End-to-end marketplace platform

Designed and developed a complete multi-role marketplace from scratch — covering database architecture, backend logic, authentication, and a fully responsive storefront.

Key Features Built

  • Role-based authentication — Admin, Seller, Customer with session management
  • Full product management — add, edit, stock tracking, image upload
  • Shopping cart backed by database (survives logout/login)
  • Checkout with JazzCash, Easypaisa & Cash on Delivery simulation
  • Order tracking timeline — pending → processing → shipped → delivered
  • Reviews & ratings — one review per verified purchase
  • Seller analytics dashboard — earnings, revenue, product performance
  • Search, category filter, price filter & sort on product listings
  • SQL injection prevention via PDO prepared statements throughout
  • Transaction-safe checkout — prevents overselling via DB-level locks

Three User Roles

🛍️ Customer
Browse, filter, add to cart, checkout, track orders, leave reviews, manage profile
🎨 Seller (Artisan)
List products, manage orders, view earnings, respond to reviews, upload documents
⚙️ Admin
Verify sellers, manage users, oversee platform — dashboard in progress
Built with: PHP MySQL PDO HTML5 CSS3 JavaScript Session Auth XAMPP

Technical Highlights

🔐
Secure Authentication
Passwords hashed with bcrypt. All queries use PDO prepared statements — SQL injection safe throughout.
🔄
DB-Backed Cart
Cart stored in MySQL, not sessions — survives logout/login. Consistent with the project's full-DB approach.
⚡
Transaction-Safe Checkout
Stock decremented inside a DB transaction with SELECT FOR UPDATE — prevents two customers overselling one item.
🛡️
Ownership Checks
Every page validates session ownership — one user cannot access another's cart, orders, or data by URL manipulation.
